# 行政区划 JSON → SQL 桌面工具

基于 **.NET WinForms** 开发，一键把四级行政区划 JSON 转换为 **SQL** 批量插入脚本。

---

## 1. 输入 JSON

```json
{
  "code": "11",
  "name": "北京市",
  "children": [
    {
      "code": "1101",
      "name": "市辖区",
      "children": [
        {
          "code": "110101",
          "name": "东城区",
          "children": [
            { "code": "110101001", "name": "东华门街道" }
          ]
        }
      ]
    }
  ]
}
```
---
## 2. 地域数据插入SQL

```
INSERT INTO public.tb_district (guid, code, "name", district_type, parent_code)
